## Ophthalmology Management Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/26ce3dc9562a4990ae0df0da55525af0/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Eye Care Products (Favorites, Add to Cart, Buy Now, Points Exchange, Like, Dislike, Comment)
5. Eye Protection Knowledge
6. Message Center
7. Personal Center (Modify Personal Information)
8. My Orders
9. My Addresses
10. My Favorites
11. My Shopping Cart
User Backend Section
12. Change Password
13. Medical Record Information
14. Follow-up Reminders
15. Appointment Booking
16. My Favorites Management
17. Order Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Member Management
4. Medical Record Information Management
5. Eye Care Products
6. Follow-up Reminder Management
7. Appointment Booking Management
8. Medication Information Management
9. Message Board Management
10. Carousel Management
11. Eye Protection Knowledge
12. Order Management
